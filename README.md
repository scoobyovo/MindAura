Project Title: **MindAura - Daily Journaling App**

`MindAura is a mood journaling app build in Android Studio using Kotlin and Jetpack Compose. Users can write daily reflections 
from selected quotes called from ZenQuotes.io, track their mood, track their daily activities and view entries through a 
custom calendar UI.`

**--FEATURES--**
1. Custom calendar view

  - Navigate between days, detect days with journal entries and view previous
   entries and reflections
3. Mood tracker

  - Select your mood for the day and track how you feel
5. Daily reflection and gratitude

  - Fill in daily gratitude entires, thoughts and notes
7. Activity selection

  - Choose activities to add context to your mood and day
8. Firebase Firestore database

  - Securely stores your journal entries and reflections
9. API quote service

  - View a list of motivation or reflective quotes, save them for later to your account and take notes

**--TECH STACK--**
Kotlin,
Jetpack Compose,
Firebase firestore & Auth,
MVVM architecture,
Material3,
Retrofit + OkHttp + Moshi,
Coroutines,

**--LOCAL SETUP--**
1. Clone repo
   'git clone *repo url*'
3. Open project in android studio
4. Add google-services.json to /app
5. Run on emulator or android device
